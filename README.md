# HW #2 Test Generation and Coverage

## Setup

    git clone https://github.ncsu.edu/pbhanda2/CSC519-HW2.git

    npm install
    node main.js
    node run test

    open coverage/lcov-report/TestGeneration/index.html
    start coverage/lcov-report/TestGeneration/index.html (Windows)
